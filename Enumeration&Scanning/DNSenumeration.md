# DNS Enumeration

--------------------------

### Dig

Enumerate:

````
dig url
````

Specific type:

````
dig type url
````

Type: NS, MX, A, etc...

-----------------------
### Host

Enumerate:

````
host url/ip
````
**ip**. If you use an ip can do a reverse lookup in order to find the domain.


Specific type:

````
host -t type url
````
